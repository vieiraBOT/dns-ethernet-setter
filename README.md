\# 🌐 DNS Ethernet Setter



Um script Python para configurar automaticamente servidores DNS em interfaces Ethernet no Windows, com teste de conexão integrado.



\## 🚀 Funcionalidades



\- 🔧 Configuração automática de DNS IPv4 e IPv6

\- 🌐 Suporte para múltiplos provedores (Cloudflare, Google, OpenDNS)

\- ✅ Teste de conexão com sites específicos

\- 🧹 Limpeza de cache DNS automática

\- 🔄 Restauração de configurações DHCP

\- 🌐 Abertura automática do site após configuração



\## 📋 Pré-requisitos



\- Windows 10 ou Windows 11

\- Acesso de Administrador

\- Python 3.6+ (já incluso no Windows 10/11)

\- Interface Ethernet conectada
